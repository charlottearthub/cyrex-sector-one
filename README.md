# Cyrex: Sector One

A mobile-first interactive RPG prototype set in the Cyrex dystopian sci-fi universe.

This project is the first test build for moving interactive web projects away from Webador and into a GitHub + Render workflow.

## Current Features

- Mobile-first game screen
- Cyberpunk/dystopian visual layout
- Health and humanity meters
- Basic story log
- Four starting player actions
- Static deployment through Render

## Live Site

https://cyrex-sector-one.onrender.com
